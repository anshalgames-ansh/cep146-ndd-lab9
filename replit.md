# CEP146 NDD Lab 9

## Overview
A Node.js/Express web application for exploring agentic AI concepts on Replit.

## Stack
- **Runtime**: Node.js 20
- **Framework**: Express 4
- **Frontend**: Static HTML/CSS (served from `public/`)

## Project Structure
- `server.js` — Express server, listens on port 5000
- `public/index.html` — Frontend landing page
- `package.json` — Node.js dependencies

## Running the App
The app runs via the "Start application" workflow:
```
node server.js
```
Serves on `0.0.0.0:5000`.
